flex-direction

This establishes the main-axis, thus defining the direction flex items areplaced in the flex container. Flexbox is (aside from optional wrapping) a single-direction layout concept. Think of flex items as primarily laying outeither in horizontal rows or vertical columns.

flex-wrap

By default, flex items will all try to fit onto one line. You can change that and allow the items to wrap as needed with this property.
